# TeoryX

AI-Powered Educational Platform

## Vision

TeoryX is a multi-tenant educational platform that combines:

- Curriculum-based learning
- AI-generated lessons
- AI tutoring
- Assessments
- Progress tracking

for students, parents and schools.
