# TeoryX MVP Backlog v0.1

## Objective

Deliver the smallest usable version of TeoryX that validates:

* AI Tutor
* Lesson Generation
* Assessment
* Progress Tracking
* Parent Visibility

---

# Deferred Features

Do not implement in MVP:

* Attendance
* Messaging
* Video classes
* Certificates
* Corporate training
* SIS integrations
* Marketplace

---

# MVP Success Criteria

Student:

* Can study
* Can chat with AI
* Can complete assessment

Parent:

* Can view progress

School:

* Can manage students

Super Admin:

* Can manage schools

If these goals are achieved, MVP is considered successful.
